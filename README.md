# Basic Calculator (Python)

This is my first Python project.

It is a simple calculator that performs basic arithmetic operations like addition, subtraction, multiplication, and division.

## Features
- Takes user input
- Supports +, -, *, /
- Beginner-friendly

## Improvements
- Handles invalid input
- Prevents division by zero
- Allows multiple calculations

## How to Run
1. Install Python
2. Run the file using:
   python calculator.py

## What I Learned
- Basic Python syntax
- Conditional statements (if-else)
- Loops
- Handling user input
